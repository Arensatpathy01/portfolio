Features Guide

A complete walkthrough of every feature — what it does, how to access it, and the tech behind it.

3D Engine AI Chatbot GitHub API Analytics

3D Engine & GLB Models

The entire background runs on Three.js r128 with a custom WebGL scene featuring wireframe geometries, neural-network-style particles, and mouse-following camera movement.

Wireframe Scene

Full-viewport WebGL canvas with rotating icosahedrons, tori, and octahedrons. Particles connect with dynamic lines to simulate a neural network.

GLB Models

Four 3D models via GLTFLoader: Robot (Hero), Damaged Helmet (Experience), Brain Stem (Projects), Astronaut (Skills). Desktop only.

Responsive 3D

3D effects progressively degrade: full on desktop, reduced on tablet, minimal on mobile, disabled on ultra-small screens.

AI Chatbot (Gemini 2.0)

Floating chatbot widget powered by Google Gemini 2.0 Flash with a complete offline fallback.

  • Open: Click the chat bubble icon (bottom-right corner on the main portfolio)
  • Online mode: Connects to Gemini API with full conversation context about Aren's background
  • Offline mode: 40+ topic keyword-matched local answers (skills, experience, projects, tech concepts)
  • Auto-fallback: If API fails (timeout, rate limit, invalid key), switches to offline answers
  • History cap: Conversation limited to 20 messages for efficiency
  • Setup: Set localStorage key gemini_api_key with your API key

Theme Switcher

Cycles through 3 modes: Dark → Light → System. Persisted to localStorage.

  • Toggle: Click the 🌙/☀️/💻 button in the sidebar footer
  • Command Palette: Type "theme" and select "Toggle Theme"
  • Light mode: Full CSS variable overrides for all cards, sidebar, hero, chatbot, forms
  • System: Follows prefers-color-scheme media query

Live GitHub Stats

Fetches live data from the GitHub REST API and renders a stats card in the About section.

  • Data shown: Public repos, stars, forks, followers, top languages (with color bar), top repositories
  • Caching: 30-minute localStorage cache to respect the 60 req/hr rate limit
  • Fallback: If API fails, shows a simplified "View Profile" card
  • Location: About section → below the info cards

Contact Form with EmailJS

Real email delivery without a backend server, powered by EmailJS (free: 200 emails/month).

  • Location: "Send Me a Query" section on the main portfolio
  • Fields: Name*, Email*, Subject, Message* (character counter: 0/1000)
  • Validation: Client-side regex email check, required field checks
  • Rate limiting: 30-second cooldown between submissions
  • Fallback: If EmailJS is not configured, opens system mail client (mailto:)
  • Setup: Replace YOUR_PUBLIC_KEY, YOUR_SERVICE_ID, YOUR_TEMPLATE_ID in js/emailService.js

Blog Section

Dynamic blog loaded from data/blog.json. Database-ready — swap the JSON URL for any API endpoint.

  • Location: Blog section (accessible from sidebar nav)
  • Features: Cover images, category badges, read-time estimates, tag pills, "Load More" pagination
  • Current posts: 4 articles (Real-Time Streaming, Career Journey, Brain Tumor Detection, Azure Certs)
  • Adding posts: Edit data/blog.json — add objects with title, excerpt, date, tags, cover, url

Dynamic Project Showcase

Projects loaded from data/projects.json instead of hardcoded HTML, with category filtering.

  • Location: Projects section on the main portfolio
  • Features: Category filter tabs (AI/ML, Systems), badges (completed/live/WIP), highlight lists, GitHub/demo links
  • Current projects: Brain Tumor Detection, Real-Time Data Streaming, Flight Data Acquisition
  • 3D tilt: All dynamically loaded cards get hover tilt effect on desktop
  • Fallback: If JSON fails to load, the static HTML fallback remains visible

Skills Radar Chart

Interactive canvas-based spider/radar chart in the Skills section.

  • Location: Skills section, below the skills grid
  • Categories: Core (C++ 92%, Python 78%…), Systems (Embedded 88%…), DevOps (Jenkins 78%…)
  • Interaction: Click tabs to switch categories. Hover data points for tooltips.
  • Animation: Animated draw-in with easeOutCubic easing

Resume Download & Tracking

Download buttons with a live counter showing total downloads.

  • Buttons: Hero CTA area ("Resume 45+") and footer download card
  • Counter: localStorage-based with a base offset of 45
  • Toast: Slide-in notification — "Resume downloading… Downloaded X+ times"
  • Analytics: Fires GA4 event resume_download
  • Setup: Place your PDF at resume/Aren_Satpathy_Resume.pdf

Custom Cursor Effects

Desktop-only animated cursor with dot, ring, glow, and canvas trail.

  • Elements: Small dot (fast), outer ring (medium), background glow (slow)
  • Trail: Canvas-based particle trail with colored particles
  • Hover: Ring expands on interactive elements (buttons, links, cards)
  • Click: Ripple animation + ring squeeze effect
  • Disabled on: Touch devices, mobile, and prefers-reduced-motion

Page Transitions & Animations

  • Split-text reveal: Hero name animates character-by-character on load (desktop only)
  • Section reveals: Cards, banners, headings fade + slide in via IntersectionObserver
  • Timeline: Progressive reveal with dot pulse animation
  • 3D parallax: Section banners shift with scroll depth
  • Glassmorphism: Backdrop-filter blur on all cards and panels
  • MutationObserver: Dynamically loaded content (projects, blog) gets animated too

Google Analytics & Consent

  • Cookie consent: GDPR-compliant banner on first visit (Accept / Decline)
  • GA4 tracking: Section views, outbound link clicks, time-on-page, form submissions, resume downloads
  • Local analytics: Events also stored in localStorage for admin visibility
  • Setup: Replace G-XXXXXXXXXX in js/analytics.js with your GA4 Measurement ID

Architecture Overview

The portfolio is a fully client-side application with no backend server required.

SEO & Accessibility

  • Meta tags: Title, description, keywords, author
  • Open Graph: og:title, og:description, og:type, og:image
  • Twitter Card: summary_large_image with title and description
  • Skip navigation: Hidden "Skip to main content" link (visible on Tab)
  • ARIA: aria-hidden on decorative 3D background, aria-label on hero
  • Reduced motion: All animations respect prefers-reduced-motion: reduce
  • Print stylesheet: Clean print output, hides interactive elements
